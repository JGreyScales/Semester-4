# SOLID + Java Review Pack (runnable mini-projects)

Source: Week-4 slides "SOLID Design Principles" (Feb 2, 2024). See the provided PDF.

## How to run (no Maven needed)
Open a terminal in *any* topic folder and run:

- Compile: `javac *.java`
- Run: `java Main`

Each folder is intentionally self-contained and uses the default package.

## What students do
Each topic includes:
- A short runnable demo in `Main.java`
- One or more starter classes
- `EXERCISES.md` describing tasks (some include TODO markers in code)

Tip: Encourage students to sketch a small class diagram for each topic before coding.
